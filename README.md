# D-tection-Automatique-de-Chats-et-Chiens-partir-d-Images-avec-le-Deep-Learning
Ce projet développe un modèle de classification d’images capable de distinguer des chats et des chiens à l’aide d’un réseau de neurones artificiels entièrement connectés (MLP). Il inclut le prétraitement des images, l’apprentissage par descente de gradient et l’évaluation des performances.
